ПК Минимальные системные требования Samp RP отображают вариант комплектации для ПК, на котором игра будет стабильно работать на минимальных настройках, не вызывая ощутимого дискомфорта.

  1. ОС: Windows 7 or later
  2. Процессор: Intel Core i5
  3. Видеокарта: NVIDIA GEFORCE GTX 750
  4. Место на диске: 200 MB
Рекомендуемые системные требования Samp RP для ПК – это ориентир того, какая мощность необходима для раскрытия полного графического и технологического потенциала игры. На комплектации, указанной в рекомендуемых требованиях, игра сможет запуститься и комфортно работать на максимальных графических настройках.
  1. ОС: Windows 10
  2. Процессор: Intel Core i7
  3. Видеокарта: NVIDIA GEFORCE GTX 970
  4. Место на диске: 200 MB
